# Sign Up App 📱

This is my first Flutter project hosted on GitHub. It’s a simple Sign-Up UI designed to practice the fundamentals of Flutter development.

## 🎯 Project Goals
The main purpose of this project was to:
* Learn and implement basic Flutter **Widgets** (TextField, Button, Column, etc.).
* Practice **UI Layout** design and responsiveness.
* Set up a professional workflow using **Git & GitHub**.

## 🛠️ Tech Stack
* **Framework:** [Flutter](https://flutter.dev)
* **Language:** [Dart](https://dart.dev)
